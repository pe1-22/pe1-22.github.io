# Website for the PE1-22 course

This is the website for the PE1-22 course at the German Sport University Cologne (DSHS). It is built using [Quarto](https://quarto.org/) and hosted on [GitHub Pages](https://pages.github.com/).

The website is available at: [https://pe1-22.github.io/](https://pe1-22.github.io/)
